# Summer-IA
Apprentissage IA été 2026 — Python, ML, Deep Learning, Vision

J'ai 19 ans, je suis un futur étudiant en Bachelor IA & Cybersécurité chez IPSSI Nice (d'ici Octobre).

Cet été j'apprends les bases de Python pour me spécialiser sur : le Machine Learning, le Deep Learning et la vision par ordinateur.
En parallèle je fais aussi : un apprentissage du trading algorithmique via la méthode ICT Sweep.

Objectif : arriver à la rentrée avec des bases solides, grâce à mon programme.

## 📚 Programme
- **Module 1** — Python fondamental : boucles, conditions, fonctions, listes
- **Module 2** — Python intermédiaire : POO, fichiers, code propre
- **Module 3** — NumPy & Pandas : manipulation de vraies données
- **Module 4** — Visualisation : Matplotlib, Seaborn
- **Module 5** — Maths codées : algèbre linéaire, descente de gradient
- **Module 6** — Machine Learning : Scikit-learn, classification, overfitting
- **Module 7** — Deep Learning à la main : réseau de neurones en pur NumPy
- **Module 8** — PyTorch : tensors, autograd, entraînement
- **Module 9** — Vision par ordinateur : OpenCV, CNN, transfer learning
